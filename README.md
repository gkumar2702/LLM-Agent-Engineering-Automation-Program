```python
prompt = f'''=== SYSTEM (IMMUTABLE) ===

'''
```

### Implement Retry Logic
```python
for attempt in range(3):
    try:
        return llm.generate(prompt)
    except Exception as e:
        if attempt == 2: raise
        time.sleep(2 ** attempt)  # Exponential backoff
```
